# أوامر البناء والتشغيل

## بناء APK (مع زيادة رقم البنية تلقائياً)

```powershell
.\build.ps1
```

## تشغيل على الويب

```bash
PUB_CACHE="F:/DevZone/pub_cache" flutter run -d chrome -t lib/main_prod.dart
```

## بناء يدوي بدون زيادة رقم البنية

```powershell
$env:PUB_CACHE="F:\DevZone\pub_cache"; $env:GRADLE_USER_HOME="F:\DevZone\gradle_cache"; flutter build apk --release -t lib/main_prod.dart
```
