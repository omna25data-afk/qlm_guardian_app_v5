# الدليل الشامل للذكاء الاصطناعي في فلاتر (Exhaustive AI & Flutter Guide)

*   **[قواعد الذكاء الاصطناعي (AI Rules)](ai-rules.md):** كيف تضبط سلوك المساعد الذكي بما يناسب مشروعك.

### المرحلة الثانية: بناء الميزات الذكية
*   **[مجموعة أدوات الذكاء الاصطناعي (AI Toolkit)](ai-toolkit.md):** بناء واجهات دردشة احترافية في دقائق.

### المرحلة الثالثة: الاحتراف والممارسات المتقدمة
