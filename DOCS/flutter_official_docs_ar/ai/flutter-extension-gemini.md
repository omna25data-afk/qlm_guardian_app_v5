# إضافة فلاتر لـ Gemini CLI (Flutter extension for Gemini CLI)

تُغير الوكلاء الذكياء (AI Agents) طريقة بناء تطبيقات فلاتر عبر المساعدة في مهام مثل بناء النماذج الأولية، مراجعة الكود، وكتابة الاختبارات. توفر **إضافة فلاتر لـ Gemini CLI** سياقاً عميقاً وأدوات متقدمة لتسريع هذه العملية.

---

## 🚀 البدء السريع (Getting Started)

تُعد هذه الإضافة (Alpha) حالياً، وهي مصممة للعمل مع Gemini CLI (إصدار 0.4.0 أو أحدث).

### التثبيت:
```bash
gemini extensions install https://github.com/gemini-cli-extensions/flutter
```

---

## 🛠️ الأوامر المتاحة

بمجرد التثبيت، ستحصل على أوامر قوية داخل جلسة Gemini CLI:

*   **`/create-app`**: يقودك عبر عملية إنشاء مشروع فلاتر جديد متوافق مع أفضل الممارسات، مع إنشاء ملفات التصميم (`DESIGN.md`) والتنفيذ (`IMPLEMENTATION.md`) قبل كتابة الكود.
*   **`/modify`**: سيبدأ جلسة تطوير موجهة لتعديل الكود الحالي، مع إنشاء خطة تعديل مقسمة لمراحل.
*   **`/commit`**: أتمتة عمليات ما قبل الالتزام (Pre-commit) مثل `dart fix` و `dart format` وتشغيل الاختبارات، مع توليد رسالة Git وصفية ذكية.

---

## 🧠 التكامل مع خادم MCP

تأتي الإضافة مهيأة تلقائياً للعمل مع **خادم Dart and Flutter MCP**. هذا يمنح Gemini القدرة على:
*   تحليل وإصلاح أخطاء المشروع.
*   التفاعل مع التطبيق المشغل (مثل Hot Reload).
*   البحث في `pub.dev` وإضافة الحزم.
*   تشغيل الاختبارات وتحليل نتائجها.

---

## 💎 بناء تطبيقات عالية الجودة
تلتزم الإضافة بقواعد صارمة تضمن أن الكود الناتج يتبع أفضل ممارسات دارت وفلاتر، بما في ذلك إمكانية الوصول (Accessibility)، التوثيق الشامل، وكتابة اختبارات الوحدة (Unit Tests).
