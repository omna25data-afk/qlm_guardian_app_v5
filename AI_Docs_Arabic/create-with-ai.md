يغطي هذا الدليل كيفية الاستفادة من أدوات الذكاء الاصطناعي لبناء ميزات مدعومة بالذكاء الاصطناعي لتطبيقات Flutter، وكذلك كيفية تبسيط وتسريع عملية التطوير في Flutter و Dart.

## نظرة عامة (Overview)

يمكن استخدام الذكاء الاصطناعي إما لبناء ميزات داخل التطبيق (مثل فهم اللغة الطبيعية وتوليد المحتوى) باستخدام حزم مثل Firebase SDK for Generative AI، أو كمساعد شخصي خلال التطوير عبر أدوات مثل Gemini Code Assist و Gemini CLI التي تعتمد على خادم Dart و Flutter MCP لتقديم سياق دقيق للكود.

## ابدأ ببناء تجارب مدعومة بالذكاء الاصطناعي في Flutter

للبدء في تطوير الميزات الذكية داخل تطبيقك، يُرشدك الدليل للموارد التالية:

* **Firebase AI Logic:** حزمة SDK الرسمية من Firebase للوصول لميزات الذكاء الاصطناعي التوليدي مباشرة في Flutter والتوافق مع Gemini أو Vertex AI.
* **Flutter AI Toolkit:** تطبيق اختباري يحتوي على مكونات (Widgets) جاهزة لبناء ميزات الذكاء الاصطناعي.

## أدوات تطوير الذكاء الاصطناعي (AI development tools)

لا يقتصر الذكاء الاصطناعي على الميزات البرمجية بل هو مساعد قوي يقلل الكود المكرر ويفهم الأكواد المعقدة. من الأدوات الموصى بها:

### GenUI SDK for Flutter

حزمة تحول المحادثات النصية إلى واجهات تفاعلية ومرئية. تعمل كمنسق بين المستخدم، مكونات Flutter، ووكيل الذكاء الاصطناعي (متاحة كنسخة Alpha).

### Antigravity

وكيل ذكاء اصطناعي مدمج في بيئة التطوير يمكنه قراءة/كتابة الكود، تشغيل أوامر الطرفية، التخطيط لمهام متعددة الخطوات، وتشغيل الاختبارات للتحقق من التغييرات.

### Gemini Code Assist

مساعد متوفر لمحررات VS Code ومحررات JetBrains و Android Studio. يساعد في:

* إكمال وإنشاء الأكواد البرمجية.
* الدردشة المدمجة في المحرر للسؤال عن مفاهيم Flutter.
* تصحيح الأخطاء (Debugging) وشرحها.

### Gemini CLI

أداة سطر أوامر تفاعلية تعمل بسلاسة دون مغادرة بيئة التطوير. يمكن من خلالها:

* إنشاء هيكل جديد لتطبيق Flutter بسرعة.
* استخدام خوادم MCP.
* أتمتة المهام كرفع التعديلات لـ Git.

#### إضافة Flutter لـ Gemini CLI

تجمع هذه الإضافة بين خادم Dart و Flutter MCP والقواعد والأوامر المخصصة (مثل `/create-app` أو `/modify`).

### خادم Dart و Flutter MCP

بروتوكول ومكمل يربط بين الذكاء الاصطناعي وأدوات مطوري Flutter لاستيعاب السياق. يتيح:

* فحص شجرة الويدجت لتصحيح الأخطاء المرئية.
* إدارة التبعيات والتعامل مع pub.dev.
* التحكم في إيقاع التشغيل (مثل Trigger hot reload).
* تحليل وحل الأخطاء المعقدة.

### قواعد لـ Flutter و Dart

ملف لإعطاء تعليمات وسياق مفصلين لنموذج الذكاء الاصطناعي حول كيفية الرد وتطبيق أفضل الممارسات البرمجية الخاصة بك.
